🛍️ Nostra - Premium Fashion E-Commerce
Nostra is a modern, responsive e-commerce landing page and product collection system built for high-end fashion brands. It features a seamless user experience, glassmorphic UI elements, and a high-performance product filtering engine.

🚀 Key Features
Dynamic Product Filtering: A "Master Filter" system that combines real-time text search and multi-category checkbox filtering.

Responsive Side-Navigation: A mobile-first navigation system with smooth transitions.

Infinite Brand Marquee: A high-performance, GPU-accelerated CSS animation for brand partnership displays.

Smooth Navigation: ID-based internal linking with scroll-behavior: smooth and fixed-header offsets.

Glassmorphic UI: Modern design language utilizing backdrop-filter and rgba transparency.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Flexbox & Grid)

Logic: Vanilla JavaScript (ES6+)

Icons: FontAwesome / Icons8

Fonts: Google Fonts (Poppins & Serif variants)
